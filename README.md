Installation
------------
If you user bundler, then just add 
```ruby
gem 'cm-sms-rails'
```
to your Gemfile and execute
```
$ bundle install
```
or without bundler
```ruby
gem install cms-sms-rails
```
Upgrade
-------
```ruby
bundle update cms-sms-rails
```
or without bundler

```ruby
gem update cms-sms-rails
```
